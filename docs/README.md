# SF08th 프로젝트 문서

## 📚 문서 목록

이 폴더에는 각 학습 카테고리별 상세 문서가 포함되어 있습니다.

### 학습 단계별 문서

1. **[01_cpp_basics.md](./01_cpp_basics.md)** - C++ 기초 문법
   - 입출력, 변수, 함수, 포인터, 참조자
   - 5개 프로젝트

2. **[02_data_structures.md](./02_data_structures.md)** - 자료구조 (STL)
   - std::vector, std::list
   - 2개 프로젝트

3. **[03_oop.md](./03_oop.md)** - 객체지향 프로그래밍
   - 클래스, 상속, 다형성, 템플릿, 스마트 포인터
   - 12개 프로젝트

4. **[04_database.md](./04_database.md)** - 데이터베이스 & SQL
   - 내장 함수, 서브쿼리, 뷰, 스토어드 프로시저
   - 12개 SQL 파일

5. **[05_network_mysql.md](./05_network_mysql.md)** - 네트워크 + MySQL
   - 소켓 프로그래밍, 멀티스레드, MySQL C++ 연동
   - 6개 프로젝트

## 🎯 학습 경로

```
C++ 기초 → 자료구조 → 객체지향 → 데이터베이스 → 네트워크/DB 통합
```

### 추천 학습 순서

1. **1주차**: C++ 기초 문법 완료
   - 변수, 함수, 포인터 이해
   - 기본 입출력 숙지

2. **2주차**: 자료구조 & 객체지향 기초
   - STL 컨테이너 활용
   - 클래스와 객체 개념

3. **3주차**: 객체지향 심화
   - 상속과 다형성
   - 템플릿과 스마트 포인터

4. **4주차**: 데이터베이스
   - SQL 기본 및 고급 문법
   - 쿼리 최적화

5. **5주차**: 실전 프로젝트
   - 네트워크 프로그래밍
   - DB 연동 애플리케이션
